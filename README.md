# Cleaning Services Website

A responsive multi-page website for a professional cleaning services business. This project is designed to showcase a service-based company offering residential and commercial cleaning solutions with a clean UI and structured user experience.

---

## Overview

This project represents a fictional cleaning services company providing multiple types of cleaning solutions, including:

* Deep home cleaning
* Soft wash services
* Pressure washing
* Window detailing
* Gutter and downspout cleaning

The website is built using static frontend technologies and focuses on presenting services in a structured and user-friendly manner.

---

## Features

* Multi-page website structure for different services
* Service-specific dedicated pages
* Quote request page
* About Us page
* Clean and simple UI layout
* Organized navigation system
* Service-based content structure
* Responsive design foundation (depending on implementation)
